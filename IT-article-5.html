# 🌈 CSS display에 대해 알아보자!
## 주로 3가지 기능을 많이 사용해요! ## 👉`inline-block;` ## 👉`block;` ## 👉`flex;` > flex는 내용이 아주 많은 관계로 다음편에서 정리해볼께요! : )

## ✅ inline-block ##### 😃엘리먼트들이 기본적으로 줄바꿈 없이 한줄에 나란히 배치되요!
👉 감싸고 있는 콘텐츠 크기에 맞춰 최소한으로 유연하게 줄어드는 특징을 가져요! ( 글자 크기에 딱 맞춰 배경색을 지정해야 할 때 주로 사용해요!)
👉 감싸고 있는 콘텐츠가 없다면 크기가 아예 사라집니다!
👉 inline의 속성을 가지고 있기 때문에 가능한 범위라면 한줄을 여럿이 함께 사용합니다!
👉 글자요소들은 기본적으로 inline 속성을 가지고 있는데, 이건 magin이나 padding을 사용할 수 없어요 ㅠ 하지만 inline-block은 가능합니다! ## ✅ block ##### 😃엘리먼트들이 마치 블럭처럼 쌓이는 모습이에요!
👉 block으로 지정된 엘리먼트들은 줄바꿈이 들어가 한 줄을 혼자 차지하는 모습을 보여요!
👉 최대한 유연하게 늘어나는 속성을 가지고 있어요! ex. a태그는 기본적으로 inline 속성이라 범위가 넓지 않아요. 이럴 때 클릭범위를 넓히기 위해서 block 속성을 지정해줍니다!
