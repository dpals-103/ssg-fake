# 🌈 CSS display:flex 공부하기 - container 속성 1탄 ## flex는 엘리먼트의 크기나 위치를 쉽게 잡아주는 도구! #### flex를 사용하기 위해서는 컨테이너 태그에 display속성을 부여해야 합니다 ``` 여기서 컨테이너란, 배치할 요소(자식)들을 감싸고 있는 큰 틀(부모) 같은 존재입니다! ```
#### 속성은 크게 2가지로 나뉩니다! ## ✅ container에 부여되는 속성 ## ✅ Item에 부여되는 속성 #### [container에 부여되는 속성] 먼저 알아볼께요!


## 📐 display #### 위에서 말했든 컨테이너 태그에 display:flex를 사용해줘야 시작할 수 있어요! #### 그래야 자식들인 flex item들을 배치할 수 있습니다!


## 📐 flex-direction #### flex-direction은 아이템들이 정렬되는 방향을 결정해요!
## `row (기본값)` #### 아이템들이 가로방향으로 배치되요! ``` 1️⃣ 2️⃣ 3️⃣ ```
## `row-reverse` #### 가로방향이지만 배치되지만 거꾸로! 배치되요! ``` 3️⃣ 2️⃣ 1️⃣ ```
## `column` #### 아이템들이 세로방향으로 배치되요! ``` 1️⃣ 2️⃣ 3️⃣ ```
## `column-reverse` #### 세로방향이지만 배치되지만 거꾸로! 배치되요! ``` 3️⃣ 2️⃣ 1️⃣ ```


## 📐 flex-wrap #### flex-wrap은 아이템들이 한 줄에 함께 있지 못할 정도로 꽉!! 차게 된다면 #### 어떤 식으로 줄바꿈을 할지 결정합니다!
## `nowrap (기본값)` #### 넘쳐서 삐져나가도 자리보존! 줄바꿈을 하지 않아요!
## `wrap` #### 컨테이너의 크기에 따라 줄바꿈을 합니다! float이나 inline-block과 비슷해요! ``` 1️⃣2️⃣ 3️⃣ ```
## `wrap-reverse` #### wrap과 비슷하지만 역순으로! 줄바꿈 합니다! 아래가 위로! 위가 아래로! ``` 3️⃣ 2️⃣1️⃣ ```
